# Decoder Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn input bytes into grammar-classified characters — a `uint32_t` key per character, derived from the YAML
grammar, with no Unicode codepoint ever assembled.

**Architecture:** `generator/chars.py` derives the character model from the grammar IR (57 named literals, 19 tested
character sets). `generator/grammar2decoder.py` emits `src/decoder_tables.h` — pure data, committed to the repository so
that building needs no Python. `src/decoder.h` holds the inline ASCII fast path; `src/decoder.c` holds the out-of-line
UTF-8 validation and the run scanner. A `make vet` sub-gate regenerates the tables and diffs them, so they cannot drift
from the grammar.

**Tech Stack:** Python 3 + PyYAML (generator), C99 (library), acutest (tests), CMake + Make (build and gate).

**Spec:** `docs/superpowers/specs/2026-07-13-decoder-design.md`. Read it before starting — it carries the rationale this
plan does not repeat.

## Task 1: The character model

Derive, from the grammar IR, the three things the tables need: the named literals, the tested sets, and the key for any
codepoint.

**Files:**

- Create: `generator/chars.py`

- [ ] **Step 1: Write the module**

A *tested set* is a **maximal** character-set node — one whose parent is not itself a character set. The ranges inside
`c-printable`'s union are constituents, not tested sets. Deduplicate by denotation.

```python
# SPDX-License-Identifier: MIT
"""The character model the decoder is built from, derived from the grammar IR.

The grammar names 57 literal characters and tests 19 distinct character sets. Every question the parser can ask about a
character is one of those 76, so `key` answers all of them at once: a 32-bit word holding the literal id, one bit per
tested set, and the character's UTF-8 length.
"""

import dataclasses
import os
import sys

sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))
import ir  # noqa: E402

MAX_CODEPOINT = 0x10FFFF

LIT_BITS = 6  # the literal id occupies bits 0..5
SET_SHIFT = 6  # the set bits start at bit 6
LEN_SHIFT = 25  # the UTF-8 length occupies bits 25..27

LIT_NONE = 0  # not one of the grammar's named characters
LIT_EOF = 58  # the sentinel keys: no set bits, so every membership test fails at them
LIT_INVALID = 59


def denote(grammar, node, seen=()):
    """The codepoint set `node` denotes as a nested tuple, or None if it is not a pure character node."""
    if isinstance(node, ir.Char):
        return ("literal", node.cp)
    if isinstance(node, ir.Range):
        return ("range", node.lo, node.hi)
    if isinstance(node, ir.Alt):
        parts = [denote(grammar, item, seen) for item in node.items]
        return None if any(part is None for part in parts) else ("union", tuple(parts))
    if isinstance(node, ir.Diff):
        base = denote(grammar, node.base, seen)
        minus = [denote(grammar, item, seen) for item in node.minus]
        if base is None or any(part is None for part in minus):
            return None
        return ("difference", base, tuple(minus))
    if isinstance(node, ir.Ref) and not node.args and node.name in grammar and node.name not in seen:
        return denote(grammar, grammar[node.name].body, seen + (node.name,))
    return None


def contains(denotation, codepoint):
    """Whether `denotation` contains `codepoint`."""
    kind = denotation[0]
    if kind == "literal":
        return codepoint == denotation[1]
    if kind == "range":
        return denotation[1] <= codepoint <= denotation[2]
    if kind == "union":
        return any(contains(part, codepoint) for part in denotation[1])
    if kind == "difference":
        return contains(denotation[1], codepoint) and not any(contains(part, codepoint) for part in denotation[2])
    raise ValueError(f"unknown denotation {denotation!r}")


def children(node):
    """The IR nodes directly nested within `node`."""
    if not dataclasses.is_dataclass(node):
        return
    for field in dataclasses.fields(node):
        value = getattr(node, field.name)
        for item in value if isinstance(value, tuple) else (value,):
            if dataclasses.is_dataclass(item):
                yield item
            elif isinstance(item, tuple):
                for nested in item:
                    if dataclasses.is_dataclass(nested):
                        yield nested


def literals(grammar):
    """Every codepoint the grammar names as a literal character, ordered by codepoint."""
    found = set()
    for production in grammar.values():
        stack = [production.body]
        while stack:
            node = stack.pop()
            if isinstance(node, ir.Char):
                found.add(node.cp)
            stack.extend(children(node))
    return sorted(found)


def tested_sets(grammar):
    """The maximal character sets the grammar tests, as an ordered `[(name, denotation)]`, deduplicated by denotation.

    A set is named for the character-set production that defines it, or, for a set the grammar tests inline, for its
    enclosing production plus an index — `ns-plain-first` holds two of them.
    """
    by_denotation = {}

    def collect(node, owner, is_inside_set):
        denotation = denote(grammar, node)
        if denotation is not None:
            if not is_inside_set and denotation[0] != "literal":
                by_denotation.setdefault(denotation, owner)
            is_inside_set = True
        for child in children(node):
            collect(child, owner, is_inside_set)

    for name, production in grammar.items():
        collect(production.body, name, False)

    # A set that a production defines outright takes that production's name.
    defined = {}
    for name, production in sorted(grammar.items()):
        denotation = denote(grammar, production.body)
        if denotation is not None and denotation[0] != "literal":
            defined.setdefault(denotation, name)

    named, inline_counts = [], {}
    for denotation, owner in by_denotation.items():
        name = defined.get(denotation)
        if name is None:
            index = inline_counts.get(owner, 0)
            inline_counts[owner] = index + 1
            name = f"{owner}-inline-{index}"
        named.append((name, denotation))
    return sorted(named)


class Model:
    """The character model: the named literals, the tested sets, and the key of any codepoint."""

    def __init__(self, grammar):
        self.literals = literals(grammar)
        self.sets = tested_sets(grammar)
        if len(self.literals) + 2 >= (1 << LIT_BITS):
            raise ValueError(f"{len(self.literals)} literals plus 2 sentinels overflow {LIT_BITS} bits")
        if SET_SHIFT + len(self.sets) > LEN_SHIFT:
            raise ValueError(f"{len(self.sets)} set bits do not fit between bit {SET_SHIFT} and bit {LEN_SHIFT}")
        self.literal_ids = {codepoint: index + 1 for index, codepoint in enumerate(self.literals)}

    def set_mask(self, index):
        """The bit mask of the tested set at `index`."""
        return 1 << (SET_SHIFT + index)

    def key(self, codepoint, length):
        """The key of `codepoint`, for a character encoded in `length` bytes."""
        key = self.literal_ids.get(codepoint, LIT_NONE) | (length << LEN_SHIFT)
        for index, (_name, denotation) in enumerate(self.sets):
            if contains(denotation, codepoint):
                key |= self.set_mask(index)
        return key

    def sentinel(self, literal_id, length):
        """The key of a sentinel: a literal id, no set bits, so every membership test fails at it."""
        return literal_id | (length << LEN_SHIFT)
```

## Task 2: Generate the decoder tables

**Files:**

- Create: `generator/grammar2decoder.py`

- Create: `src/decoder_tables.h` (its output, committed)

- [ ] **Step 1: Write the generator**

The non-ASCII keys must collapse to exactly five constants — NEL, BOM, the C1 control zone, the noncharacters, and one
shared "ordinary content character". The generator **asserts** this: if a grammar change ever broke the collapse, the
hand-written ladder in `decoder.c` would silently misclassify, so generation must fail loudly instead.

```python
# SPDX-License-Identifier: MIT
"""Emit the decoder's tables from the grammar IR — the data half of the decoder.

The UTF-8 mechanics are fixed by RFC 3629 and live hand-written in `decoder.c`; only the classification is grammar-
derived, and all of it is here.

Usage: `python3 generator/grammar2decoder.py > src/decoder_tables.h`
"""

import os
import sys

sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))
import chars  # noqa: E402
import spec2grammar  # noqa: E402

NEL = 0x85
BOM = 0xFEFF
NONCHARACTERS = (0xFFFE, 0xFFFF)
C1_ZONE = tuple(cp for cp in range(0x80, 0xA0) if cp != NEL)

# Codepoints that must all share the single "ordinary content character" key: everything above ASCII that UTF-8 can
# encode and that is neither a named literal nor an excluded oddity.
def content_codepoints():
    """Representatives of every non-ASCII segment expected to share the content key."""
    return (0xA0, 0xD7FF, 0xE000, 0xFDD0, 0xFEFE, 0xFF00, 0xFFFD, 0x10000, 0x10FFFF)


def identifier(name):
    """The C identifier for a grammar name: `ns-plain-safe-in` becomes `NS_PLAIN_SAFE_IN`."""
    return name.upper().replace("-", "_")


def utf8_length(codepoint):
    """The number of bytes UTF-8 uses to encode `codepoint`."""
    if codepoint < 0x80:
        return 1
    if codepoint < 0x800:
        return 2
    if codepoint < 0x10000:
        return 3
    return 4


def emit(out, model):
    """Write the whole header."""
    out.write("// SPDX-License-Identifier: MIT\n")
    out.write("// Generated by generator/grammar2decoder.py from the vendored yaml-grammar. Do not edit.\n")
    out.write("#ifndef YEAST_DECODER_TABLES_H\n#define YEAST_DECODER_TABLES_H\n\n")
    out.write("#include <stdint.h>\n\n")

    out.write("// The named literal characters. Id 0 means the character is not one of them.\n")
    for codepoint in model.literals:
        out.write(f"#define YS_LIT_{codepoint:04X} {model.literal_ids[codepoint]}u\n")
    out.write(f"\n#define YS_LIT_EOF {chars.LIT_EOF}u\n")
    out.write(f"#define YS_LIT_INVALID {chars.LIT_INVALID}u\n\n")

    out.write("// One bit per character set the grammar tests. Unions and subtractions are already evaluated here.\n")
    for index, (name, _denotation) in enumerate(model.sets):
        out.write(f"#define YS_SET_{identifier(name)} 0x{model.set_mask(index):08X}u\n")
    out.write("\n")

    out.write("// Set ids, for ys_scan_set().\n")
    out.write("typedef enum ys_set_id {\n")
    for name, _denotation in model.sets:
        out.write(f"    YS_SET_ID_{identifier(name)},\n")
    out.write("    YS_SET_ID_COUNT\n} ys_set_id;\n\n")

    out.write("// The mask of each set, indexed by its id.\n")
    out.write("static const uint32_t YS_SET_MASK[YS_SET_ID_COUNT] = {\n")
    for index, (name, _denotation) in enumerate(model.sets):
        out.write(f"    YS_SET_{identifier(name)},\n")
    out.write("};\n\n")

    out.write("// The sentinel keys: a literal id and no set bits, so every membership test fails at them.\n")
    out.write(f"#define YS_KEY_EOF 0x{model.sentinel(chars.LIT_EOF, 0):08X}u\n")
    out.write(f"#define YS_KEY_INVALID 0x{model.sentinel(chars.LIT_INVALID, 1):08X}u\n\n")

    out.write("// The key of every ASCII character, including its length. Index it with the byte.\n")
    out.write("static const uint32_t YS_ASCII[128] = {\n")
    for codepoint in range(0x80):
        out.write(f"    0x{model.key(codepoint, 1):08X}u,\n")
    out.write("};\n\n")

    out.write("// The non-ASCII keys, without the length bits: decoder.c ORs the length in.\n")
    out.write(f"#define YS_KEY_NEL 0x{model.key(NEL, 0):08X}u\n")
    out.write(f"#define YS_KEY_BOM 0x{model.key(BOM, 0):08X}u\n")
    out.write(f"#define YS_KEY_C1_CONTROL 0x{model.key(C1_ZONE[0], 0):08X}u\n")
    out.write(f"#define YS_KEY_NONCHARACTER 0x{model.key(NONCHARACTERS[0], 0):08X}u\n")
    out.write(f"#define YS_KEY_CONTENT 0x{model.key(content_codepoints()[0], 0):08X}u\n\n")

    out.write("// The length bits of a key.\n")
    out.write(f"#define YS_LENGTH_BITS(length) ((uint32_t)(length) << {chars.LEN_SHIFT})\n\n")
    out.write("#endif // YEAST_DECODER_TABLES_H\n")


def check_collapse(model):
    """Fail loudly unless the non-ASCII keys collapse to the five constants decoder.c's ladder assumes."""
    for group, expected in (
        (C1_ZONE, model.key(C1_ZONE[0], 0)),
        (NONCHARACTERS, model.key(NONCHARACTERS[0], 0)),
        (content_codepoints(), model.key(content_codepoints()[0], 0)),
    ):
        for codepoint in group:
            actual = model.key(codepoint, 0)
            if actual != expected:
                raise ValueError(
                    f"U+{codepoint:04X} has key {actual:#010x}, not {expected:#010x}: the non-ASCII keys no longer "
                    f"collapse, so the classification ladder in decoder.c is wrong"
                )


def main():
    model = chars.Model(spec2grammar.load())
    check_collapse(model)
    emit(sys.stdout, model)


if __name__ == "__main__":
    main()
```

- [ ] **Step 1: Write the check**

```python
# SPDX-License-Identifier: MIT
"""Gate the committed decoder tables against the grammar.

Regenerates `src/decoder_tables.h` and diffs it against the committed copy, and checks the model's invariants: the key
must pack without collision, and each set bit must agree with a direct evaluation of the set's definition over every
codepoint. Exits non-zero on any mismatch.
"""

import io
import os
import subprocess
import sys

sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))
import chars  # noqa: E402
import grammar2decoder  # noqa: E402
import spec2grammar  # noqa: E402

TABLES = "src/decoder_tables.h"


def regenerated():
    """The header as the grammar says it should be, clang-formatted exactly as the committed copy is."""
    model = chars.Model(spec2grammar.load())
    grammar2decoder.check_collapse(model)
    buffer = io.StringIO()
    grammar2decoder.emit(buffer, model)
    formatted = subprocess.run(
        ["clang-format", f"--assume-filename={TABLES}"],
        input=buffer.getvalue(),
        capture_output=True,
        text=True,
        check=True,
    )
    return formatted.stdout


def check_keys(model):
    """Every set bit of every codepoint's key must agree with a direct evaluation of the set's definition."""
    errors = []
    for index, (name, denotation) in enumerate(model.sets):
        for codepoint in range(chars.MAX_CODEPOINT + 1):
            is_set = (model.key(codepoint, 1) & model.set_mask(index)) != 0
            if is_set != chars.contains(denotation, codepoint):
                errors.append(f"U+{codepoint:04X}: key disagrees with {name}")
                break
    return errors


def main():
    model = chars.Model(spec2grammar.load())
    errors = check_keys(model)
    with open(TABLES, encoding="utf-8") as handle:
        if handle.read() != regenerated():
            errors.append(f"{TABLES} is stale: regenerate it with `python3 generator/grammar2decoder.py`")
    if errors:
        for error in errors:
            print(error, file=sys.stderr)
        sys.exit(1)
    print(f"decoder tables OK: {len(model.literals)} literals, {len(model.sets)} sets")


if __name__ == "__main__":
    main()
```

`tests/test_decoder.c`:

```c
// SPDX-License-Identifier: MIT
#include "acutest.h"
#include "decoder.h"
#include <stdbool.h>
#include <stdint.h>

// An empty window is end of input: no set bits, so every membership test fails at it, and no test site anywhere in the
// parser needs an end-of-input special case.
static void test_end_of_input(void) {
    ys_char character = ys_next_char((const uint8_t *)"", 0);
    TEST_CHECK(YS_LIT(character) == YS_LIT_EOF);
    TEST_CHECK(YS_LEN(character) == 0);
    TEST_CHECK((character & YS_SET_NS_CHAR) == 0);
    TEST_CHECK((character & YS_SET_C_PRINTABLE) == 0);
}

// An ASCII character is one table load: its literal id, the sets it belongs to, and a length of one.
static void test_ascii_character(void) {
    ys_char dash = ys_next_char((const uint8_t *)"-x", 2);
    TEST_CHECK(YS_LEN(dash) == 1);
    TEST_CHECK((dash & YS_SET_C_INDICATOR) != 0);
    TEST_CHECK((dash & YS_SET_NS_CHAR) != 0);
    TEST_CHECK((dash & YS_SET_S_WHITE) == 0);

    ys_char space = ys_next_char((const uint8_t *)" ", 1);
    TEST_CHECK((space & YS_SET_S_WHITE) != 0);
    TEST_CHECK((space & YS_SET_NS_CHAR) == 0); // ns-char is c-printable minus the whitespace

    ys_char digit = ys_next_char((const uint8_t *)"7", 1);
    TEST_CHECK((digit & YS_SET_NS_DEC_DIGIT) != 0);
    TEST_CHECK((digit & YS_SET_NS_HEX_DIGIT) != 0);

    ys_char nul = ys_next_char((const uint8_t *)"\0", 1);
    TEST_CHECK(YS_LIT(nul) == 0);                // not a named literal
    TEST_CHECK((nul & YS_SET_C_PRINTABLE) == 0); // and not printable
    TEST_CHECK(YS_LEN(nul) == 1);                // but its length tells it apart from end of input, which has none
}

// The only individually named characters above ASCII: NEL and the byte-order mark.
static void test_named_non_ascii(void) {
    ys_char nel = ys_next_char((const uint8_t *)"\xC2\x85", 2);
    TEST_CHECK(YS_LEN(nel) == 2);
    TEST_CHECK((nel & YS_SET_C_PRINTABLE) != 0);

    ys_char bom = ys_next_char((const uint8_t *)"\xEF\xBB\xBF", 3);
    TEST_CHECK(YS_LEN(bom) == 3);

    TEST_CHECK(YS_LIT(nel) != 0 && YS_LIT(bom) != 0);

// Every codepoint UTF-8 can encode reports the right length, and every surrogate is rejected. The encoder here is
// deliberately independent of the decoder — that is what makes this a check rather than a tautology.
static void test_every_codepoint(void) {
    for (uint32_t codepoint = 0x80; codepoint <= 0x10FFFF; codepoint++) {
        uint8_t bytes[4];
        size_t size;
        if (codepoint < 0x800) {
            bytes[0] = (uint8_t)(0xC0u | (codepoint >> 6));
            bytes[1] = (uint8_t)(0x80u | (codepoint & 0x3Fu));
            size = 2;
        } else if (codepoint < 0x10000) {
            bytes[0] = (uint8_t)(0xE0u | (codepoint >> 12));
            bytes[1] = (uint8_t)(0x80u | ((codepoint >> 6) & 0x3Fu));
            bytes[2] = (uint8_t)(0x80u | (codepoint & 0x3Fu));
            size = 3;
        } else {
            bytes[0] = (uint8_t)(0xF0u | (codepoint >> 18));
            bytes[1] = (uint8_t)(0x80u | ((codepoint >> 12) & 0x3Fu));
            bytes[2] = (uint8_t)(0x80u | ((codepoint >> 6) & 0x3Fu));
            bytes[3] = (uint8_t)(0x80u | (codepoint & 0x3Fu));
            size = 4;
        }

        ys_char character = ys_next_char(bytes, size);
        bool is_surrogate = codepoint >= 0xD800u && codepoint <= 0xDFFFu;
        if (is_surrogate) {
            if (!TEST_CHECK(YS_LIT(character) == YS_LIT_INVALID)) {
                TEST_MSG("U+%04X: surrogate accepted", codepoint);
                return;
            }
        } else if (!TEST_CHECK((size_t)YS_LEN(character) == size)) {
            TEST_MSG("U+%04X: length %u, expected %zu", codepoint, (unsigned)YS_LEN(character), size);
            return;
        }
    }
}

// SPDX-License-Identifier: MIT
#ifndef YEAST_DECODER_H
#define YEAST_DECODER_H

#include "decoder_tables.h"
#include <stddef.h>
#include <stdint.h>

// A classified character: the grammar's answer to every question it can ask about one, packed into a word.
//
//   bits  0..5   the named-literal id — 0 for "not a named literal", or YS_LIT_EOF / YS_LIT_INVALID
//   bits  6..24  one bit per character set the grammar tests; the masks are the YS_SET_* constants
//   bits 25..27  the number of input bytes the character consumed, 0..4
//
// No Unicode codepoint is ever assembled. Nothing in the parser compares a character's numeric value — tokens are spans
// of input bytes — so the decoder validates and classifies rather than decoding.
typedef uint32_t ys_char;

// The named-literal id of a character, or 0 if it is not one of the grammar's named characters.
#define YS_LIT(character) ((character) & 0x3Fu)

// The number of input bytes a character consumed.
#define YS_LEN(character) (((character) >> 25) & 0x7u)

// The mask of the character set `set`.
static inline uint32_t ys_set_mask(ys_set_id set) {
    return YS_SET_MASK[set];
}

// Classify the character at the head of the window. Bytes >= 0x80 only; ys_next_char() handles ASCII inline.
ys_char ys_next_char_slow(const uint8_t *bytes, size_t size);

// Advance while the character is in `set`, returning the number of bytes consumed.
size_t ys_scan_set(const uint8_t *bytes, size_t size, ys_set_id set);

// The next character at the head of a window of `size` readable bytes, or YS_KEY_EOF when the window is empty.
//
// The `size == 0` branch is predicted not-taken almost always, so at true end of input the processor mispredicts once
// and speculatively loads the byte past the buffer to index YS_ASCII. That is Spectre-v1 shape but not a threat: the
// address is always exactly `bytes + size`, with no attacker-controlled offset, so it cannot be steered into an
// arbitrary read. Architecturally the bounds check is correct. Hardening it would cost a barrier per character, or a
// padded-input contract that ys_new_string_parser() cannot impose on a caller's buffer.
static inline ys_char ys_next_char(const uint8_t *bytes, size_t size) {
    if (size == 0) {
        return YS_KEY_EOF;
    }
    if (bytes[0] < 0x80) {
        return YS_ASCII[bytes[0]];
    }
    return ys_next_char_slow(bytes, size);
}

#endif // YEAST_DECODER_H
```

// SPDX-License-Identifier: MIT
#include "decoder.h"
#include <stdbool.h>

// What a lead byte says about its sequence: how many bytes it takes, and the legal range of the byte that follows it.
// That range is what rejects overlong encodings (E0 admits only A0..BF), surrogates (ED admits only 80..9F) and
// codepoints beyond U+10FFFF (F4 admits only 80..8F) — with no codepoint arithmetic at all. A length of 0 marks a byte
// that cannot lead a sequence.
typedef struct ys_lead {
    uint8_t length;
    uint8_t first_min;
    uint8_t first_max;
} ys_lead;

// The UTF-8 shape of a lead byte. Fixed by RFC 3629; nothing in the grammar can change it.
static ys_lead lead_of(uint8_t byte) {
    if (byte >= 0xC2u && byte <= 0xDFu) {
        return (ys_lead){2, 0x80u, 0xBFu};
    }
    if (byte == 0xE0u) {
        return (ys_lead){3, 0xA0u, 0xBFu}; // below A0 the sequence would be overlong
    }
    if (byte == 0xEDu) {
        return (ys_lead){3, 0x80u, 0x9Fu}; // above 9F lie the surrogates, which UTF-8 cannot encode
    }
    if (byte >= 0xE1u && byte <= 0xEFu) {
        return (ys_lead){3, 0x80u, 0xBFu};
    }
    if (byte == 0xF0u) {
        return (ys_lead){4, 0x90u, 0xBFu}; // below 90 the sequence would be overlong
    }
    if (byte == 0xF4u) {
        return (ys_lead){4, 0x80u, 0x8Fu}; // above 8F lies everything past U+10FFFF
    }
    if (byte >= 0xF1u && byte <= 0xF3u) {
        return (ys_lead){4, 0x80u, 0xBFu};
    }
    return (ys_lead){0, 0, 0}; // 80..BF are continuations, C0/C1 are always overlong, F5..FF encode nothing
}

// Whether `byte` is a UTF-8 continuation byte.
static inline bool is_continuation(uint8_t byte) {
    return (byte & 0xC0u) == 0x80u;
}

// Classify the character at the head of the window. Bytes >= 0x80 only; ys_next_char() handles ASCII inline.
//
// A sequence running past the end of the window is invalid, which is the right answer at true end of input. A stream
// buffer that hands us a partial sequence before its true end has a bug of its own; there is no "incomplete" answer.
ys_char ys_next_char_slow(const uint8_t *bytes, size_t size) {
    const ys_lead lead = lead_of(bytes[0]);
    if (lead.length == 0 || size < (size_t)lead.length) {
        return YS_KEY_INVALID;
    }
    if (bytes[1] < lead.first_min || bytes[1] > lead.first_max) {
        return YS_KEY_INVALID;
    }
    for (size_t index = 2; index < (size_t)lead.length; index++) {
        if (!is_continuation(bytes[index])) {
            return YS_KEY_INVALID;
        }
    }

    // Above ASCII the grammar names only NEL and the byte-order mark, and excludes from c-printable only the C1
    // controls and the two noncharacters. Everything else that is valid is ordinary content, whatever its length — the
    // generator asserts that collapse, so this ladder cannot silently fall out of step with the grammar.
    uint32_t key = YS_KEY_CONTENT;
    if (lead.length == 2 && bytes[0] == 0xC2u) {
        if (bytes[1] == 0x85u) {
            key = YS_KEY_NEL;
        } else if (bytes[1] < 0xA0u) {
            key = YS_KEY_C1_CONTROL;
        }
    } else if (lead.length == 3 && bytes[0] == 0xEFu) {
        if (bytes[1] == 0xBBu && bytes[2] == 0xBFu) {
            key = YS_KEY_BOM;
        } else if (bytes[1] == 0xBFu && bytes[2] >= 0xBEu) {
            key = YS_KEY_NONCHARACTER;
        }
    }
    return key | YS_LENGTH_BITS(lead.length);
}

// Advance while the character is in `set`, returning the number of bytes consumed. The ASCII branch is the hot one and
// the vectorization target: a nibble-table lookup classifies sixteen bytes at a time under SSSE3 or NEON, and drops in
// behind this same signature without the generated parser changing a line.
size_t ys_scan_set(const uint8_t *bytes, size_t size, ys_set_id set) {
    const uint32_t mask = ys_set_mask(set);
    size_t offset = 0;
    while (offset < size) {
        if (bytes[offset] < 0x80) {
            if ((YS_ASCII[bytes[offset]] & mask) == 0) {
                break;
            }
            offset += 1;
        } else {
            const ys_char character = ys_next_char_slow(bytes + offset, size - offset);
            if ((character & mask) == 0) {
                break;
            }
            offset += YS_LEN(character);
        }
    }
    return offset;
}

## What is deliberately not here

- **SIMD kernels.** `ys_scan_set` ships scalar, behind its final signature. The vector kernels — nibble tables, `pshufb`
  / `vqtbl1q_u8`, runtime dispatch — drop in behind the same call once a benchmark exists to prove they win. The
  generated parser never changes.
- **Buffer management.** How a stream parser fills, grows and pins buffers so token spans survive a refill. The decoder
  is a pure function of a `(bytes, size)` window precisely so this stays a separate decision.
- **The parser.** `grammar2parser.py` consumes this. It is the next body of work, not this one.
