# Skills

Skills we use on this project.

## Core loop

- `superpowers:test-driven-development` — red/green/refactor; write the test before the code.
- `superpowers:systematic-debugging` — root-cause any bug or oracle divergence before proposing a fix.
- `superpowers:verification-before-completion` — no "done" without running the command and showing output.

## Planning

- `superpowers:writing-plans` — turn a design/spec into a step-by-step implementation plan.
- `superpowers:executing-plans` — run a written plan with review checkpoints.
- `superpowers:subagent-driven-development` — execute independent plan tasks via subagents in-session.

## Review

- `superpowers:requesting-code-review` — request review when a step is complete, before merge.
- `superpowers:receiving-code-review` — verify and reason about feedback, don't blind-apply.
- `superpowers:code-reviewer` — the review agent itself.
